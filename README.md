# hello-ryan-02

헬로 라이언 ZOOM 라이브 **ep02 2부 '나만의 가상 브랜드'** 실습 키트 마켓플레이스입니다. 70대 할아버지도 따라 만드는 완전 초보용.

## 설치 방법

Claude Code 안에서 아래 두 줄을 차례로 칩니다.

```
/plugin marketplace add yoonjechoi/hello-ryan-02
/plugin install brand-kit@hello-ryan-02
```

## 쓰는 법

- 설치가 끝나면 **"내 가상 브랜드 만들어줘"** 라고 말하면 인터뷰부터 시작합니다.
- 인터뷰 → 회사명·로고 → 디자인 시트(내 브랜드 스킬) → 산출물 4종까지 단계별로 함께 만듭니다.
- 한 번 만든 브랜드는 **"내 브랜드로 카드뉴스 만들어줘"** 처럼 다시 불러 쓸 수 있습니다.

## 포함된 스킬

- **brand-interview** — 따뜻한 인터뷰로 브랜드 재료(하고 싶은 일·꿈·청중)를 끌어냅니다.
- **brand-sheet** — 레퍼런스 사이트를 보고 색·폰트·로고를 정해 '내 브랜드 스킬'로 패키징합니다.
- **build-brand-kit** — 인터뷰부터 웹·PPT·카드뉴스·명함까지 한 번에 만드는 오케스트레이터입니다.
