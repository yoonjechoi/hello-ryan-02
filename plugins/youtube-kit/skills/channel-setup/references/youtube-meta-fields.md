# 유튜브 채널 메타 필드 · 이미지 규격표 (youtube-meta-fields)

`02_channel-setup.md`에 채울 **값 목록**과 **이미지 규격·프롬프트 템플릿**. 규격 수치는 **그대로** 쓴다(임의 변경 금지).

## 채워 넣는 값 (텍스트)
| 필드 | 규칙 | 비고 |
|---|---|---|
| **채널명** | 짧고 기억 쉽게, 주제가 연상되게 | 후보 3개 → 참가자가 택1 |
| **핸들(@핸들)** | 영문·숫자·밑줄(_)·마침표(.) / 3~30자 | 채널명 로마자 축약. 중복 대비 대안 2개 |
| **채널 설명** | 2~4문장. 무엇을·누구에게·왜 | 첫 문장에 핵심(검색 노출) |
| **키워드/태그** | 주제 키워드 5~10개, 쉼표 구분 | 주제·청중·지역에서 |
| **카테고리** | 유튜브 기본 카테고리 중 1개 | 추천 + 이유 한 줄 |
| **방향성 한 줄** | "___한 사람에게 ___를 주는 채널" | 메인에서 고른 방향성 반영 |

> 핸들은 우리가 '사용 가능'을 확정할 수 없다 → **대안 2개**를 함께 주고 "유튜브에서 초록 체크(사용 가능)를 확인하세요"라고 안내한다. 없는 '사용 가능' 단정 금지.

## 채널 이미지 규격 (정확 수치 — 보존)
| 용도 | 권장 크기 | 안전영역 / 비고 |
|---|---|---|
| **프로필 사진(아바타)** | **800×800 px** | 원형으로 표시 → 가장자리 여유. 작게도 또렷하게 |
| **채널 아트(배너)** | **2560×1440 px** | **모든 기기 공통 안전영역 1546×423 px** — 텍스트·로고는 이 안에. 밖은 기기 따라 잘림 |
| **영상 썸네일** | **1280×720 px (16:9)** | 최소 너비 **640 px**. 일관된 색·글자 위치 규칙으로 채널 통일감 |

## 이미지 프롬프트 템플릿 (영어 복붙 + 한국어 설명)

### 프로필(아바타) — 800×800
```
[brand/person symbol or simple portrait], centered, clean flat background,
bold and simple, recognizable when small, square, no small text
```
- 한국어 설명: "작게(원형) 떠도 한눈에 알아볼 단순한 상징/얼굴. 글자 거의 없이."

### 배너(채널 아트) — 2560×1440, 안전영역 1546×423
```
[channel theme] banner, key text and logo kept inside the center safe area,
calm background extending to edges, [mood/colors], 2560x1440, wide cinematic
```
- 한국어 설명: "채널명·한 줄은 **가운데 안전영역(1546×423)** 안에. 바깥은 배경만(잘려도 되게)."

### 썸네일 스타일 가이드 — 1280×720
```
youtube thumbnail style, 1280x720, big readable title area on one side,
consistent accent color, face or key object on the other side, high contrast
```
- 한국어 설명: "앞으로 모든 썸네일이 **같은 규칙**(글자 자리·포인트색·얼굴/사물 위치)이게. 작게 떠도 글자가 읽히게 큰 글씨·강한 대비."

## 사실·안전 규칙
- 규격 px·안전영역 수치는 위 표 그대로. 모르면 비워 두고 추측 수치 금지.
- 이미지에 실제 상표·로고·유명인 얼굴을 흉내 내지 않는다(일반 상징·사물·본인 사진 우선).
- 카테고리·필드 이름이 유튜브 화면과 다르게 보일 수 있으니, "유튜브 스튜디오 → 맞춤설정"에서 비슷한 항목을 찾으라고 한 줄 덧붙인다.
