# 채널 세팅 (channel-setup)

`01_video-plan.md`(영상 기획·인터뷰 재료)를 **재사용**해, 내 유튜브 채널에 그대로 채워 넣을 **모든 값 + 채널 이미지 프롬프트**를 만든다. 영상 한 편과 채널이 **같은 결**로 보이게 맞추는 단계다.

> 구글 아이디가 있으면 채널은 이미 있다 → **메타(채널 정보)만 채우면 끝.** (참가자 약 80~90%가 아직 미세팅.) 새 영상의 톤·주제·청중을 채널 얼굴에도 입힌다.

---

## 두 갈래로 일한다 (선택=메인 / 생성=서브)

- **방향성·레퍼런스 선택은 메인 대화(실시간)** — `01_video-plan.md`를 바탕으로 **채널 방향성 후보 2~3개**와 (있으면)레퍼런스 채널 결을 제시하고, **참가자가 하나를 고른다.** 반드시 답을 기다린다.
  - ⚠️ **선택을 서브에이전트에 넘기지 말 것.** 서브에이전트는 참가자와 실시간 대화를 못 해 "방향성 골라라"를 못 묻고 **자동 선택돼 버린다(brand-kit v0.0.3에서 겪은 실제 버그).** 고른 결과만 서브에이전트에 넘긴다.
- ⚠️ **생성은 서브에이전트(opus)로** — 채널명·핸들·설명·태그·카테고리 도출 + **이미지 프롬프트(프로필·배너·썸네일)** 를 `Agent(subagent_type:"channel-setup-maker", model:"opus")`가 만든다. → 산출 `02_channel-setup.md`.
- 데이터는 **파일로 전달**: `_workspace/youtube-kit/<slug>/`.

---

## 만들 값 (모두 01의 주제·청중·톤에서 도출)
필드 목록·규칙 상세: `references/youtube-meta-fields.md`.
- **채널명** — 기억하기 쉽고, 주제가 떠오르게. 후보 3개를 주고 고르게 한다.
- **핸들(@핸들)** — 영문·숫자·밑줄·마침표, 3~30자. 채널명을 로마자로 짧게. (중복이면 끝에 숫자·단어를 붙여 대안 2개 더.)
- **채널 설명(소개)** — 무엇을·누구에게·왜 보면 좋은지 2~4문장. 첫 문장에 핵심(검색에도 잡힘).
- **키워드/태그** — 채널 주제 키워드 5~10개(쉼표 구분). 주제·청중·지역에서.
- **카테고리** — 유튜브 기본 카테고리 중 가장 가까운 하나(추천 + 이유 한 줄).
- **채널 방향성 한 줄** — "이 채널은 ___한 사람에게 ___를 주는 곳" 한 문장(메인에서 고른 방향성 반영).

## 만들 이미지 프롬프트 (영어 프롬프트 + 한국어 설명 + 정확 규격)
규격표·프롬프트 템플릿: `references/youtube-meta-fields.md`.
1. **채널 프로필 사진(아바타)** — 권장 **800×800 px**(원형으로 표시되니 가장자리 여유). 단순·또렷하게(작게도 알아보게).
2. **채널 아트(배너)** — 권장 **2560×1440 px**, 모든 기기 공통 **안전영역(텍스트·로고) 1546×423 px**. 안전영역 밖은 기기에 따라 잘린다.
3. **영상 썸네일 스타일 가이드** — 유튜브 권장 **1280×720 px(16:9), 최소 너비 640 px.** 색·글자 크기·얼굴/포인트 위치의 **일관 규칙**을 정해, 앞으로 모든 영상이 한 채널로 보이게.

> ⚠️ 이미지 프롬프트는 **영어(복붙용) + 한국어 설명** 한 쌍. 규격(px·안전영역)은 위 숫자를 **그대로** 쓴다 — 임의로 바꾸지 않는다.

## 게이트 + 교육
- 정리된 `02_channel-setup.md`(값 + 이미지 프롬프트)를 **보여주고** "이 채널 얼굴 맞아요? (이름·핸들·방향성 고칠 데?)" 확인 — 통과 전 마무리로 안 넘어간다.
- 채널에 **어디에 붙여 넣는지**도 한 줄씩 안내(채널명·핸들·설명은 유튜브 스튜디오 '맞춤설정 → 기본 정보', 프로필·배너는 '브랜딩').
- 교육 안내: *"채널 세팅을 다시 받고 싶으면 '내 유튜브 채널 세팅해줘', 이름만 바꾸려면 '채널명 다시 지어줘'라고 치면 돼요."*

## 사실·보존 규칙
- `01_video-plan.md`에 없는 경력·수치·고유명사 금지. 규격 수치(800×800·2560×1440·1546×423·1280×720)는 보존.
- 핸들 중복 여부는 우리가 확정할 수 없으니, **대안 2개를 함께** 주고 "유튜브에서 사용 가능 표시를 확인하세요"라고 안내(없는 '사용 가능' 단정 금지).

## 깎기 포인트 (v1 → 개선 후보)
- 채널명 후보 품질이 핵심 → 약하면 작명 패턴 예시를 더 준다.
- 썸네일 스타일 가이드가 막연하면, 영상 1편 썸네일 예시 1장을 같이 그려 기준을 보여준다.
