# 디자인 시트 추출 레시피 (design-extraction)

레퍼런스 사이트의 인상을 **토큰**으로 옮겨 `color-typography-specs.md`를 만든다. 목표: 네 산출물(웹·PPT·카드뉴스·명함)이 **그대로 복붙**해 같은 옷을 입는 것.

## 1. 색 팔레트 (hex 고정)
사이트에서 받은 인상을 아래 역할로 정리한다. **포인트색은 딱 1개** (CTA·핵심 강조).
| 역할 | 예시 토큰 | 메모 |
|---|---|---|
| 주색 | `--primary` | 헤드라인·로고·강조 바탕 |
| 보조 | `--primary-2` | |
| 배경 | `--bg` | 페이지(살짝 따뜻하거나 차갑게) |
| 표면 | `--surface` | 카드 |
| 본문 | `--ink` / `--ink2` | 텍스트·보조텍스트 |
| **포인트** | `--accent` | **화면당 한 곳만** (떡칠 금지) |
| 라인 | `--line` | 헤어라인 |

> 색은 사이트 픽셀이 아니라 **인터뷰 무드**에서 정하는 게 기본. 널리 아는 브랜드면 대표색 근사 가능, 그 사실을 한 줄 표기.

**포인트색 규칙:** 주색과 **다른 색상 계열**에서 고른다(주색 초록→주황/팥색, 갈색→청록/남색). 같은 계열은 강조가 묻힌다. 흰 글자 얹는 버튼은 충분히 어둡게. **어두운 배경 위 강조어는 '밝은 포인트 변형'을 따로** 둔다: `.card.dark .accent{color:<밝은 변형>}`.

**어두운 바탕(무드가 '어두움')일 때:** `--bg`/`--surface`를 차콜 계열로, 본문은 밝은 글자(`#f2f5f4`)·보조는 뮤트 그레이로 뒤집는다. **그림자는 어두운 면에서 안 보이니 카드 분리는 1px 라인 + 미세 글로우**로. (밝은 배경 전용으로 굳히지 말 것.)

## 2. 타이포 — 무드로 가른다 (명조 기본값 금지)
제목 폰트는 무드(인터뷰 7번 '글씨 느낌')로 정한다. **한모금이 명조라고 다 명조로 재탕하지 말 것.**

| 무드 | 제목 폰트(구글/CDN, 무료) |
|---|---|
| 따뜻·고급·클래식·정갈 | `Noto Serif KR` · `Gowun Batang` · `Nanum Myeongjo` (명조) |
| 단단·공예·슬랩 | `Gowun Batang`(700) + 라틴만 `Roboto Slab` |
| **모던·강렬·에너지·스포츠·테크** | **`Black Han Sans` · `Jua` · `Do Hyeon` · `Pretendard` 800/900 (굵은 산세리프)** |

- **본문/UI:** 항상 `Pretendard`.
- ⚠️ 한글 전용 무료 '슬랩세리프'는 사실상 없다 → 무게 있는 명조로 대체, 라틴만 Roboto Slab을 얹는다.
- 스케일·행간 1줄로 명시(H1/H2/본문/캡션).

## 3. 컴포넌트 규칙
카드(라운드·그림자·라인), 버튼(주=포인트색·라운드), 배지/태그, 여백 원칙, 1~2개 모티프(절제).

## 4. 산출 — 복붙용 CSS 블록 (필수)
`color-typography-specs.md`에 **그대로 붙여 쓰는 `<style>` 블록**을 포함한다:
- 폰트 `<link>`(Pretendard + 제목 폰트)
- `:root` 토큰 전체(위 hex) — 어두운 바탕이면 `--accent`의 밝은 변형도 포함
- 공통 클래스 `.btn .tag .card` + `--serif/--sans/--shadow` 변수
- **워드마크 락업 스니펫**(logo-wordmark.md 산출) — 이걸 빠뜨리면 산출물마다 로고가 들쭉날쭉

> 골드 예시: `brand-skill-template.md`의 한모금 `color-typography-specs.md` 참고. 그 구조를 브랜드 색·폰트만 바꿔 재사용.

## 일관성 비결
이 `:root` 토큰 + 컴포넌트 CSS를 **네 산출물 파일에 동일하게** 넣는다 → 웹·PPT·카드뉴스·명함이 한 브랜드로 보인다.
